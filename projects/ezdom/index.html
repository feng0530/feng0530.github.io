EZ-DOM 一起動

2025-03-23
實作練習
Java • Spring Boot • Java Script

專案介紹 Link to heading

專案期間: 2023/07 ~ 2023/09

擊破現代科技帶來的冷漠與疏離感， 我們致力於創建一個平台， 讓人們輕鬆建立社交連結和結交新朋友。 透過運動討論、活動和專業教練課程的結合， 在運動的同時，與他人共享快樂、學習新知識。 在這裡，你不再是孤單的一個人， 無聊就揪，揪了就動，EZ-DOM一起動!!

專案連結

專案概述 Link to heading

  • 使用Maven建構的Spring Boot專案
  • Spring Boot: 2.7.x
  • Java: 17
  • 個人負責項目:
    1. 教練專區
    2. 預約教練
    3. 聯絡客服

技術應用 Link to heading

前端技術 Link to heading

  1. 利用BootStrap5網站模板進行頁面設計。
  2. 利用JavaScript進行網頁的資料驗證、實現動態效果。
  3. 利用Fetch API向後端發送請求，根據回應資料動態渲染頁面。

後端技術 Link to heading

  1. 遵循REST風格實現RESTful API。
  2. 使用Spring Data JPA實現資料的CRUD操作。
  3. 使用Fetch API向後端API發送請求。
  4. 使用WebSocket實現會員與後台管理員之間的即時客服功能。
  5. 使用Redis存取即時客服的對話紀錄，增加資料存取的效率。
