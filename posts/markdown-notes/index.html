3.Tabgroup: Link to heading 

{{< tabgroup >}}
{{< tab name="Hello" >}}
Hello World!
{{< /tab >}}

{{< tab name="Goodbye" >}}
Goodbye Everybody!
{{< /tab >}}
{{< /tabgroup >}}

效果如下:

Hello World!
Goodbye Everybody!

向右對齊 Link to heading 
{{< tabgroup align="right" >}}
{{< tab name="Hello" >}}
Hello World!
{{< /tab >}}

{{< tab name="Goodbye" >}}
Goodbye Everybody!
{{< /tab >}}
{{< /tabgroup >}}

效果如下:

Hello World!
Goodbye Everybody!

展示不同程式碼 Link to heading

Tabgroup支援任何 Markdown語法

{{< tabgroup style="code" >}}
{{< tab name="Ruby" >}}

```ruby
puts 'Hello'
```

{{< /tab >}}
{{< tab name="Python" >}}

```python
print('Hello')
```

{{< /tab >}}
{{< tab name="JavaScript" >}}

```js
console.log("Hello");
```

{{< /tab >}}
{{< /tabgroup >}}

效果如下: 

puts 'Hello'

print('Hello')

console.log("Hello");

Mermaid.js Link to heading

Mermaid.js 是一個可以在網頁中使用 Markdown 語法來繪製圖表的 JavaScript 庫。它的主要用途是讓開發人員和文件撰寫者，用 簡單的語法 快速生成 流程圖、時序圖、甘特圖 等視覺化圖表。

如果你想在你的網站上使用Mermaid-JS， 可以在你的markdown file使用Shortcode
